# Indiedu

## Detail Project

Indiedu is a website that provides a service where students can obtain learning media independently through a website or application platform.

## Technology Used

Here are the technologies I use:

- Figma - for creating design stuff like UI, logo, etc.
- Javascript - main programming language
- VueJS (NuxtJS) - framework for creating the website
- TailwindCSS - for styling the website
- Netlify - hosting platform for the website
