## Problem

Given an integer array `nums`, return `true` if any value appears at least twice in the array, and return `false` if every element is distinct.

## Example

### Example 1:

```bash
Input: nums = [1,2,3,1]
Output: true
```

### Example 2:

```bash
Input: nums = [1,2,3,4]
Output: false
```

## Answer

```py
import collections

nums = [1, 2, 3, 1]
get = [item for item, count in collections.Counter(nums).items() if count > 1]
if len(get) > 0:
    print(True)
else:
    print(False)

```
